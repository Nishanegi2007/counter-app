## Features

- **Increment** — increase the count by 1
- **Decrement** — decrease the count by 1
- **Reset** — bring the count back to 0
- 
---

## Getting Started

No dependencies, no build step. Just open the file.

```bash
git clone https://github.com/your-username/counter-app.git
cd counter-app
open index.html
```

Or simply download the ZIP and open `index.html` in your browser.

## Usage

| Button | Action             |
|--------|--------------------|
| `+`    | Adds 1 to counter  |
| `−`    | Subtracts 1        |
| Reset  | Resets to 0        |
